HTML

  • The head elements contains information about the webpage, such as the language it is written in
  • The body element represents the visible content shown to the user
  • While the head content is not shown to users, it contains metadata that helps the browser render the page correctly
  • You must use an opening tag and a closing tag to define each element
  • The style ement defines the CSS styles associated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as the character set, description, key words, author, and viewport
  • Elements nested inside other elements are called child elements
  • Semantic HTML elements make your HTML easier to read by other developers and the browser

CSS

  • Cascading Style Sheets is a programming language used to define how webpage content is presented to users
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class attribute allows us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
